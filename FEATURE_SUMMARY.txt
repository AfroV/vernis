WHAT'S DIFFERENT?
═══════════════════════════════════════════════════════════════════════════

IPFS GUI has:
   • Tkinter GUI with buttons and progress bars
   • Visual file selection
   • Completion dialog boxes
   • "Open File Location" button

Vernis Advanced has:
   • Same core engine (IPFSManager, downloader logic)
   • Command-line interface
   • Web API integration
   • Designed for Raspberry Pi/server use

Think of it as:
   IPFS GUI = Desktop app for manual use
   Vernis Advanced = Server daemon for automated use

EXAMPLE: NESTED DOWNLOAD
═══════════════════════════════════════════════════════════════════════════

Input: bafybeigdyrzt5sfp7udm7hu76uh7y26nf3efuylqabf3oclgtqy55fbzdi

Step 1: Download CID
   ⬇ bafybei...di → metadata.json

Step 2: Parse JSON, find nested CIDs
   {
     "image": "ipfs://baf456...",     ← Found!
     "animation": "ipfs://baf789...",  ← Found!
     "attributes": [...]
   }

Step 3: Download nested CIDs
   ⬇ baf456... → image.png
   ⬇ baf789... → animation.mp4

Result: 3 files downloaded automatically!

Both IPFS GUI and Vernis Advanced do this identically.

DEPLOYMENT NOTES
═══════════════════════════════════════════════════════════════════════════

1. Both downloaders are copied to /opt/vernis/scripts/ during install

2. Flask API uses advanced downloader by default:
   /opt/vernis/scripts/nft_downloader_advanced.py

3. Advanced downloader auto-detects IPFS:
   • Checks common ports automatically
   • Falls back to public gateways
   • No configuration needed!

4. Workers default to 2 (good balance):
   • Not too aggressive for Pi's network
   • Faster than single-threaded
   • Can increase via API: workers=4

MIGRATION FROM IPFS GUI
═══════════════════════════════════════════════════════════════════════════

If you have existing IPFS GUI CSV files:

1. Copy CSV to Vernis:
   scp my-collection.csv pi@vernis.local:/opt/vernis/uploads/

2. Run advanced downloader:
   ssh pi@vernis.local
   cd /opt/vernis
   python3 scripts/nft_downloader_advanced.py \
      --csv uploads/my-collection.csv \
      --workers 4

3. Or upload via web UI:
   http://vernis.local/add.html
   → Drag & drop CSV
   → Same result!

Your existing CSVs work without modification!
